# 高级day1-作业

## Description

读取10个整型数据12 63 58 95 41 35 65  0 38 44，然后通过归并排序，对该组数据进行排序，输出有序结果，每个数的输出占3个空格

## Input

> 12 63 58 95 41 35 65  0 38 44

## Output

> 0 12 35 38 41 44 58 63 65 95

## Sample Input 1

```text
12 63 58 95 41 35 65  0 38 44
```

## Sample Output 1

```text
  0 12 35 38 41 44 58 63 65 95
```
